# Задание 3 — Параллельный поиск минимума и максимума с использованием OpenMP
## Условие задания

Реализовать поиск минимального и максимального элементов массива из 1 000 000 чисел с использованием OpenMP и сравнить время выполнения с последовательной версией.

## Пояснение решения

В параллельной версии массив обрабатывается несколькими потоками. Каждый поток вычисляет локальные значения минимума и максимума для своей части массива.

После завершения вычислений локальные результаты объединяются в общие переменные. Для корректной работы используется директива critical, которая предотвращает одновременный доступ потоков к общим данным и исключает гонки данных.

Время выполнения параллельной версии сравнивается с временем последовательного алгоритма, после чего рассчитывается ускорение.
