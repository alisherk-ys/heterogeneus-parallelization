# Задание 2 — Последовательный поиск минимального и максимального элементов
## Условие задания

Создать массив из 1 000 000 элементов, найти минимальное и максимальное значения последовательным способом и измерить время выполнения алгоритма.

## Пояснение решения

Создаётся динамический массив, который заполняется случайными числами. Минимальное и максимальное значения инициализируются первым элементом массива.

Далее выполняется последовательный проход по массиву, в процессе которого происходит сравнение каждого элемента с текущими значениями минимума и максимума.

Для измерения времени выполнения используется функция omp_get_wtime(). Время вычисляется как разница между моментом начала и окончания работы цикла.
