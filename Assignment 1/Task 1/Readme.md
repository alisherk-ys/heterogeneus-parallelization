# Задание 1 — Динамический массив и вычисление среднего значения
## Условие задания

Необходимо создать динамический массив из 50 000 целых чисел, заполнить его случайными значениями, вычислить среднее значение элементов массива и корректно освободить выделенную память.

## Пояснение решения

Для хранения данных используется динамический массив, так как его размер задаётся во время выполнения программы. Память выделяется с помощью оператора new[].

Массив заполняется случайными числами от 1 до 100. Для вычисления среднего значения сначала считается сумма всех элементов массива, после чего сумма делится на количество элементов. Чтобы избежать переполнения, сумма хранится в переменной типа long long.

В конце выполнения программы память освобождается с помощью delete[], что предотвращает утечку памяти.
