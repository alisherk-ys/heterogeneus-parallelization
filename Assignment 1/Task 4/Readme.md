# Задание 4 — Вычисление среднего значения с использованием reduction
## Условие задания

Создать массив из 5 000 000 чисел, вычислить среднее значение последовательным и параллельным способом с использованием директивы reduction, а затем сравнить время выполнения.

## Пояснение решения

Последовательная версия реализована путём суммирования всех элементов массива в одном цикле с последующим делением на размер массива.

В параллельной версии применяется директива reduction. Каждый поток вычисляет свою локальную сумму элементов, после чего OpenMP автоматически объединяет все частичные суммы в итоговое значение. Такой подход позволяет избежать конфликтов при работе с общей переменной и обеспечивает более высокую производительность по сравнению с использованием critical.
