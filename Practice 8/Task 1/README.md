# Задание 1 — CPU: обработка массива с использованием OpenMP

## 1. Цель задания

Нужно реализовать программу для **параллельной обработки массива на CPU** с
использованием **OpenMP**.

По заданию требуется:

- создать массив размером `N` (в примере `N = 1 000 000`),
- реализовать обработку массива (умножение каждого элемента на 2),
- замерить **время выполнения обработки на CPU**.


## 2. Идея решения

Обработка массива является **data-parallel задачей**:

- каждый элемент массива обрабатывается **независимо** от остальных,
- порядок выполнения не важен,
- это идеально подходит для параллелизации через `#pragma omp parallel for`.

OpenMP автоматически:
- создаёт пул потоков,
- делит итерации цикла `for` между потоками,
- синхронизирует завершение цикла.


## 3. Подготовка данных

В программе:

1) создаётся вектор `data` размером `N`,
2) массив инициализируется значениями индексов:
   - `data[i] = i`

Это удобно для проверки корректности, так как результат после умножения легко предсказать.


## 4. Параллельная обработка с OpenMP

### Основная директива
```cpp
#pragma omp parallel for
```

Она означает:

- цикл for будет выполнен параллельно,

- итерации автоматически распределяются между потоками,

- каждый поток работает со своей частью массива.

Внутри цикла выполняется операция:

`data[i] *= 2.0;`


Так как каждый поток пишет в разный элемент массива, гонок данных не возникает, и синхронизация не требуется.

## 5. Замер времени выполнения

Время измеряется с помощью `std::chrono::high_resolution_clock:`

1. перед параллельным циклом фиксируется start_time,

2. после завершения цикла фиксируется end_time,

3. разница переводится в миллисекунды.

Измеряется именно:

- время обработки массива (параллельная часть).

## 6. Проверка корректности результата

После выполнения программы выводятся несколько элементов массива:

- `data[0]`

- `data[N/2]`

- `data[N-1]`

Так как изначально:

- `data[i] = i`

после обработки ожидается:

- `data[i] = 2 * i`

Это позволяет быстро убедиться, что вычисления выполнены корректно.

## 7. Что выводит программа

Программа печатает:

- размер массива `N`,

- время выполнения обработки с использованием OpenMP,

- несколько элементов массива для проверки корректности.
