# Задание 1 — Сумма массива на CUDA (глобальная память)
## 1. Цель задания

Нужно реализовать CUDA-программу, которая вычисляет сумму элементов массива размером 100 000, используя глобальную память GPU (т.е. данные лежат в global memory и читаются оттуда в kernel).
Затем требуется сравнить:

- результат суммы (CPU vs GPU),

- время выполнения (последовательный CPU vs GPU kernel).

## 2. Идея решения

Решение состоит из двух частей:

### CPU-версия (последовательно)
На процессоре обычным циклом суммируем все элементы массива.

### GPU-версия (CUDA kernel)

- Массив копируется на GPU в глобальную память (cudaMalloc, cudaMemcpy).

- Kernel запускается с сеткой GRID блоков и BLOCK потоков в блоке.

- Каждый блок суммирует “свой” кусок массива:

  - каждый поток читает один элемент из глобальной памяти d_in[global_i],

  - складывание внутри блока выполняется через редукцию в shared memory,

  - один поток (tid=0) записывает сумму блока в глобальный массив d_blockSums.

- После kernel на CPU считываются d_blockSums и выполняется финальная сумма (последний шаг редукции уже на CPU).

Такой подход соответствует заданию: основные данные читаются из global memory, а ускорение достигается за счёт параллельности и быстрой редукции в пределах блока.
